# NexusAI Solutions — Pricing Tiers

## Pricing Philosophy
Transparent, value-based pricing with no hidden fees. All tiers include core platform access — you pay for scale, support level, and advanced features.

---

## Starter Tier
**For small teams getting started with AI**

**$2,500/month** (billed annually) or $3,000/month (billed monthly)

### Included
- Up to 5 users
- 2 AI modules (choose from: Predictive Analytics, Document Intelligence, Recommendations, Conversational AI)
- 10,000 API calls/month
- 50GB data storage
- 3 pre-built connectors
- Cloud SaaS deployment only
- Email support (48-hour SLA)
- Community access and documentation

### Best For
- Startups and small businesses
- Proof-of-concept projects
- Single department deployments

---

## Professional Tier
**For growing companies scaling AI across teams**

**$8,500/month** (billed annually) or $10,000/month (billed monthly)

### Included
- Up to 25 users
- All 5 AI modules
- 100,000 API calls/month
- 500GB data storage
- 10 pre-built connectors + custom connector builder
- Cloud SaaS or Dedicated Cloud deployment
- Priority email + chat support (8-hour SLA)
- Dedicated Customer Success Manager
- Monthly performance reviews
- A/B testing framework
- Custom model training (up to 3 models)
- SSO/SAML integration

### Best For
- Mid-market companies (100-1,000 employees)
- Multi-department rollouts
- Companies with moderate data volumes

## Enterprise Tier
**For large organizations with complex requirements**

**Custom pricing** — typically $25,000-$75,000/month based on scope

### Included
- Unlimited users
- All 5 AI modules + early access to beta features
- Unlimited API calls
- Unlimited data storage
- All connectors + custom integrations with dedicated engineering support
- Any deployment option (Cloud, Dedicated, Hybrid, On-Premises)
- 24/7 phone + chat support (1-hour critical SLA)
- Dedicated Solutions Engineer
- Quarterly Business Reviews with executive team
- Unlimited custom model training
- Advanced compliance suite (HIPAA, SOC 2, GDPR, PCI-DSS)
- Multi-region deployment
- Custom SLA agreements
- Professional services hours included (implementation, training, optimization)

### Best For
- Enterprise organizations (1,000+ employees)
- Highly regulated industries (Healthcare, Financial Services, Government)
- Global deployments requiring data residency
- Organizations requiring on-premises deployment
